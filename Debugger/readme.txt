Tex.Asm/Debugger
=======
Примитивный отладчик.
Есть два файла mdeb и mdeb2, это для трассировки и для брейкпоинта соответственно.
Два файла r1 и r3 - обработчики 1 и 3 прерываний соответственно. Это все резиденты, они останутся в памяти. То есть вначале запускаем r1 и r2, потом все остальное.
Отлаживается файл HW1
HW1.lst нужен для проверки правильности вывода ip.

И в случае брейкпоинта, и в случае трассировки на экран лишь выводится ip
